Imprimindo Esse página HTML

Nessa aula veremos como imprimir uma pagina sem ter que ter no na folha de impressão, todos os dados que se mostra no site, somente o que é ideal de ser impresso

Por onde começamos a separar os tipos?

Para começar, devemos montar nosso html normal, e criar duas folhas de estilo, uma para impressora e outro pro site, mas como assim para impressora? no caso vamos criar um css que vai mostrar somente a vizualização para impressão, removendo algumas informações ou adicionando.

dentro do css do site devemos colocar depois do href="nome.css" midia="screen", que seria basicamente vizualização para telas, sem descriminar qual tamanho ainda, e o css de impressora colocamos href="nome.css" midia="print", para indicar que todo css de impressão vai ser feito dessa forma.

Estilo de impressão

No estilo de impressão podemos colocar somente oque queremos mostra para imprimir, com isso partes como menu, ou rodapé de sites, videos e outros, podemos tirar, para isso, dentro do css de impressora, indicamos a tag, e colocamos display:none: que como vimos nos outras aulas que já tivemos, ele basicamente não mostra na tela.
podemos criar uma informação que só que aparecer somente na impressão, exemplos " esse artigo foi feito pelo site marcelo.com.br"
