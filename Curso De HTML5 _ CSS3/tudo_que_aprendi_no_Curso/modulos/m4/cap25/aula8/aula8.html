Css para cada tamanho de dela

Para as midias queries, podemos usar um limitador de tela, que identifica qual o tamanho de tela em que o usuario está e mostra o site de acordo com a tela, para isso funcionar devemos fazer varias declarações de css, porem uma para cada tamanho médio de tela, que pode varias entre umas 5.

E como fazemos isso?
Como já vimos, agora podemos usar o metedo mobile firts, com isso sabemos que ja criamos o css para celular, agora iremos fazer para outras midias, nesse caso criamos um outro css, e dentro usaremos o paremetro @media screen and (min-width: 000px) and (max-width: 000px){declarações d css}, essa parametro significa que as declarações que estiver dentro dele, será aplicada do tamanho x até o tamanho x, caso atingir o tamanho maximo mudar para o css mais próximo, ou se não tiver fica naquele mesmo, como mostra na imagem abaixo.
