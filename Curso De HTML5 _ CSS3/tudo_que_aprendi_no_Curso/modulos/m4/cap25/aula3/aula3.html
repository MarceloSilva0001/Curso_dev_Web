Midia Query

A Midia Query é igual a Midia Type e Midia Features

  • Midia Type Tipo de Midia, que seriam telas em Geral, sem distinção de tamanho
  • Midia Features Caracteristica de Midia, que seria telas por tamanho, como de celulares, notebook, telão, smartWatch
