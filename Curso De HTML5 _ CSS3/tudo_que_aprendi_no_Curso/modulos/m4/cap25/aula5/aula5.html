Reunindo tudo em um único CSS

exercicio
Anotação

Nessa aula veremos como reunir os arquivos de css para midia query todos em um só, o ideal é sempre usar os 3 arquivos de css, pois para manutenção do código, fica muito melhor.

Dentro desse mesmoa arquivo Css, iremos usar 2 declarações que são @media screen and (orientation: portrait){ colcar os dados de estilo} declarações para o modo retrato, e o @media screen and (orientation:landscape){colcar os dados de estilo} para o modo paisagem.

Dessa forma, declaremos os dois css, indicando onde será o tipo de tela com isso para projetos mais simples podemos usar apenas 1 css, com as declarações indicando o modo de vizualização.
