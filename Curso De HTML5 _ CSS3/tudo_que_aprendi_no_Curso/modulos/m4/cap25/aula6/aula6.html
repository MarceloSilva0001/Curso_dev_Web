Mobile First

O que é mobile first?
Mobile first é um projeto que faz a criação de projetos web e sites primeiro em dispositivos móveis, para depois fazer adaptações para o desktop e outras plataformas, ao contrário do que ainda acontece na maioria das vezes.

Porque o mobile first é importante?
O mobile first é uma estratégia bastante utilizada no planejamento da arquitetura de sites. O objetivo é garantir que ele rode bem em dispositivos móveis, como é o caso dos smartphones e tablets. Na prática, o desenvolvedor projeta, primeiramente, o site em telas menores para depois adaptá-las para as maiores
