Seguindo a orientação do dispositivo

Exercicio
Anotação

Podemos usar duas forma inciar para as informações se adpatar a diferença de tela, nesse Exercicio, iremos usar um formato que é bem usado nos celulares, paisagem e retrato, para isso vamos usar 3 arquivos css, um chamado estilo ou outros dois de retrato e paisagem, dentro de cada uma vamos passar o termo

midia="" e dentro as aspas colar o ,modo de exibição

Modo Geral

No Modo geral, criamos um arquivo CSS, passando o parametro media="all", que vai se tratar de todas as telas

Modo retrato

No css retrato, colocamos o parametro media="screen and (orientation: portrait)" que como ja vimoos "screen" indica telas, mas agora informado a orientação da tela em modo retrato

Modo Paisagem

No css Paisagem, colocamos o parametro media="screen and (orientation: landscape)" que como ja vimoos "screen" indica telas, mas agora informado a orientação da tela em modo Paisagem

Dentro de cada css, iremos colocar as declarações de acordo com a orientação da tela, com isso sempre que o usuario trocar o modo de tela, ele não vai ter o site quebrado.
