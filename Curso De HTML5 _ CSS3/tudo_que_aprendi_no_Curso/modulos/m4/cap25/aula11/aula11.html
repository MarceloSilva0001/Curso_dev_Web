Menu se escondendo ao click

Para esse efeito acontecer, foi preciso usar javascrip, criando um script, o profressor criou uma função e deu a ela o nome de clickMenu, com essa função, sempre que o usuario clica nos 3 pontinhos, o menu vai se recolher(o menu é uma div no modo semantico).

Primeiros passos, para criar o menu, pegamos um icone de menu, o famoso menu 3 traços, como pegamos ele do google, usamos um link exteno de css que é mostrado na aula, dentro da tag desse icone, demos um id ela ja vem com class por padrão, dentro ainda dessa tag, no final colamos uma parametro chamado onclick="função" e dentro do campo que está escrito função, foi criado uma função chamada clickMenu Essa função foi criada na hora mesmo, usando javascript, dentro da tag "menu" foi dado um id chamado itens com isso apontamos o o bloco que deveria ser escondido. No resumo a função clickMenu, vai retornar um display: block; quando o usuario clica no icone de hambuerguer, e o menu vai aparecer, e para esconder, quando o usurio clica vai retornar display:none; Para o menu não aparecer

Função clickMenu

Como tinha falado no texto anterior, essa função foi criada na hora para se encaixar a funcionalidade, era funciona da seguinte forma.

Podemos criar um documento externo ou interno fora de uma div, mas dentro do body.
Primeiro criamos um Script e dentro desse scripit criamos a função com o nome que quisermos o nome desse foi o clickMenu, então dentro do script fica assim

script
function clickMenu(){
if(itens.style.display == 'block'){
itens.style.display = 'none'
}else{
itens.style.display = 'block'
}
}

Basicamente essa função tem a seguinte lógica de programação, se itens.style.display for igual a block, o itens.style.display recebe o parametro 'none' que significa esconder, se n~]ao estiver, ele rebecebe 'block'
