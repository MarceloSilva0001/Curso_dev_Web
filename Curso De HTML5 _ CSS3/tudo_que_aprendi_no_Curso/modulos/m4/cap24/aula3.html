Métodos GET e POST para formulários

Anotação

GET

<form action="cadastro.php">

Existem dois metados para envio de formulario, quando não colocamos metodos algum, por padrão usamos o metodo get, GET ele envia os dados direto na url, faça um teste de envio após criar o forumulario, depois quando der erro de envio ou enviar, veja na url os dados que você colocou.
O metodo get também só pode ser usado caso não ultrapassar 3.000 caracteres.

POST

<form action="cadastro.php" method="post" autocomplete="off">

quando criamos o forumulario usando o metodo post, os dados somem da url, aparecem somente o submit de envio("cadastro.php") do formulario, porem ainda sim não está seguro, caso inspecionar o código, ainda sim é possivel validar os dados.
Caso quiser proteger, será preciso enviar os dados por https, porem é uma forma mais avançada que leva mais tempo para aprender, no momento esses dois exemplos servem.

Quando usar? o metodo get pode ser usado quando não se envia informações importantes como (dados bancarios, informações de senha e identidade) o metodo post, é usado nesses casos pois esconde esses dados.
