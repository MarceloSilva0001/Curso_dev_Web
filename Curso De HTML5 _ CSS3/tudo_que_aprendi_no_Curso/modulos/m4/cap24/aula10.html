Lista suspensa com nomes fixos

Podemos usar uma lista, de escolha, como escolha de estado, para listar listar a opções usamos a tag select dentro dela colocamos a opções que são informadas com a tag option dentro de "option" colocamos os nomes das escolhas, como ex, cidades ou estados.
Uma das opções que podemos fazer e sepração de grupo, nesse exemplo serparmos os estados por regisão, pata isso selecionamos as regiões, e usamos a tag optgroup que separa as regiões

Lista com opção de digitar nomes ou selecionar

Na lista de prossisões, fizemos um campo que possibilita o usuario tanto selecionar quanto digitar o nome de sua profissão, para isso o processo basicamente começa, crindo uma label e um input do tipo texto, dentro da tag dos mesmo, colocamos uma tag chamada datalist e dentro da tag colocamos as opções, quem usam o formato de tag option também, para que essa lista seja informada dentro do datalistdevemos colocar o atributo id="nome da lista" e dentro da tag d input, colocar o atributo list="nome da lista" com isso o suario consegue digital ou selecionar dentro desse campo.
