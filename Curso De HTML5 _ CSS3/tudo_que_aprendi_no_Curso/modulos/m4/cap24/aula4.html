Criando caixas de texto e de senha

Campo de preenchimento Obrigatório

Podemos tornar um campo de preenchimento obrigatório, dessa forma o formulario não será enviado sem dados faltando, dentro da tag de input colocamos no final o comando riquired para força o preenchimento.

Quantidade minima e máxima de letras

Dentro ainda da tag de input colocamos a função minlength="quantidade minima" EX: ="5".
Para coloca quantidade maxima usando também dentro da tag input a função maxlength="5"

Campo de preenchimento com informação

Dentro da caixa podemos inficar oque deve ser preenchido, colocando a função placeholder="nome do usuario" essa função é colocado dentro da tag de input olhe os exemplos abaixo.

Completação automatica de campos

Quando preenchemos dados em vario sites, muitas vezes os navegadores salvam nossas informações, com isso geralmente quando vamos preencher um formulario em um site os dados como nome, numero telfone ou até senha de login, salvam, para que nosso site possa dar essa facilida ao usuario, usamos a função de autocomplete="on" dentro da tag de formulario, e para o naveador informar corretamente o dadom dentro da tag de preenchimento input devemos identificar oque vai ser preenchido, no caso de um campo de nome colocamos autocomplete="username" campo de senha autocomplete="new-password", porem o próprio VS code mostra todas as opções de auto complete que podemos usar.
