Nesse ex de CSS, vemos que no cabeçalho, no lugar da tag de "style" foi um outra forma para trazer o css para a página, usando um link externo do arquivo que contem o css todo declarado <.link rel="stylesheet" href="style.css">, se abrirmos o arquivo de css, poderemos ver que colocamos uma regra para formatação de caracteres especiais(@charset "UTF-8";), com essa forma de css externa podemos de forma mais simples dar manutenção nas paginas, pois como vai estar pelo link o css, vai ser aplicado de forma geral em todas as páginas que usem o link, e caso quiser, você pode criar mais de uma declaração de css e usar dois links no cabeçalho
