Lista de links externos

Criamos nessa aula, o estilo do aside, que dentro continha listas, essa lista foi separda em duas lista uma de cada lado, colocamos icones de corrente simbolizando que aquele link é de um site externo ao nosso, e colocamos icones de lista de uma forma geral sem precisar ficar coloando um á um.

Para colocar icones em todas os links das lista, usamos a funnção list-style-type:'\Código do icone';

Para colocar um icone depois do link, e não antes, coloquei class em todos os link externos, e usar o after, que indica que depois do elemento faça tal ação (.NomeDadoAclass::after{ content:'\Código do icone'; }) e dentro do css chamei a função.
