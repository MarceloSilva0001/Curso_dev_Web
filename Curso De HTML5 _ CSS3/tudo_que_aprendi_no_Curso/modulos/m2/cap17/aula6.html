Variáveis em CSS

Nessa aula vamo usar Variaveis, para facilitar no uso das cores

-Para declarar variaveis em Css, colocamos dentro do bloco ":root{--nomeVariavel: conteudo;}" como fizemos no mini projeto,

exemplo::root{--fonte-padrao: arial,cursiva, sans serif}
div{ font-family: var(--fonte-padrao) }

Foi mostrado como declarar dentro de de uma função alguns informações como margin 0 para todos as caixas sem precisar tirara uma a uma.
