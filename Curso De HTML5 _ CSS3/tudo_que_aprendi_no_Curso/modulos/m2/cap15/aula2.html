As Diferençs entre id e Class

Dentro do código de HTML, não podemos usar o mesmo ID para mais de uma tag, funciona, mas é errado, com isso nossa página fica com aqueles erros de exeção quando apertamos f12 , porem porem se quisermos usar a mesma configuração usamos "class" dentro da tag, pois dessa forma podemos aplicar a mesma configuração sem fazer da forma errada.

ID , usamos dessa forma no html: "< H1 id="x1" >" , dessa forma eu moutro que dentro desse H1 recebeu esse id, e dentro do css vai ficar assim "#x1{ color:white;}", aqui eu disse que o ID x1 vai ter a cor branca, porem não podemos usar esse mesmo ID x1 para outras tag, se formos usar podemor criar uma Class


Class, usamos dessa forma no HTML, "< H1 class="x1" >", dessa forma eu moutro que dentro desse H1 recebeu essa class, e dentro do css vai ficar assim ".x1{ color:white;}", aqui eu disse que a class "x1" vai ter a cor branca, porem já podemos usar essa mesma para mais de uma tag, e com isso aplicando essa informação para mais tags
