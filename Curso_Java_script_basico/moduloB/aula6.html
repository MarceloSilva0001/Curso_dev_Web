Tratamento de dados

Coletando nome e mostrando no alerta

para fazer esse processo criamos uma variavel, e atribuimos o dado coletado no aletar. exemplo:
Coletar o nome: var nome = window.prompt("Digite seu nome")
Mostrar o nome: window.alert('é um prazer em te conhecer,'+ nome) temos o primeiro texto dentro de aspas simples, e depois dela vem o "+" e o nome da variavel pra mostrar na tela o dado armazenado naquela variavel

Colentando dados para soma

Devemos converter os numeros que vão ser somados, mas porque temos que converter? pois de forma automatica o java script nesse comando de coleta dos dados, trasnforma todos carcteres em string, porem podemos indicar como devera ser feita essa coleta, quando se tratar de tipos Primitivos diferentes

Number.Float(n) Para coverter numero real

Dessa Forma, quando formos coletar algum dado com pelo colocamos o comando dessa forma: Number.parseInt(window.prompt("Ano de nascimento?")) atras do "prompt("Ano de nascimento?")" vem o dado de conversão.
