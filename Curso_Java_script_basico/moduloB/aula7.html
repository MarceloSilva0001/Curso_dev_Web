Operadores Aritméticos

+, -, *, /, %, **

+ : Soma, Exemplo: 5+2=7

- : Subtração, Exemplo: 5-2= 3

* : Multiplicação, Exemplo: 5*2= 10

/ : Divisão Real, Exemplo: 5/2 = 2.5

% : Resto da divisão inteira,Exemplo: 5 % 2= 1, Oque seria isso? quando fazemos um calculo de divisão, sempre chega um momento as vezes de ter que colocar uma "," para continuar a o calculo, e nessa parte sempre sobre o resto, no entanto, não iremos continuar, ficamos com o valor do resto da divisão inteira, Exemplo 10 / por 2 o resto 0

** : potência, Exemplo: 5 ** 2+ 25

Ordem de procedência

A ordem de procedência, seria é a ordem em que o operador para a primeira operaçãos ser realizada, exemplo: na calc 5 + 3 / 2, o calc vem primeiro no operador "/", com isso podemos mudar a ordem, colocando parenteses, (5+3) / 2, com isso o calc vai ser pirmeiro no operador "+", depois pula para o operador "/"
